01 · 토론 아젠다

처형·부여에 맞는 메타를 어떻게 짜나?

"스텟은 뱀서를 그대로 베끼되 안 맞는 건 빼고, 신개념 스텟 하나를 더할까?
스킬은 뱀서처럼 무기·장신구(자동·패시브)로 짤까?"

지금은 강화 카드 16종이 임시 방식(스텟·스킬 체계 미정). 처음으로 틀을 잡는 결정이다.

질문 ①

스텟 개념

뱀서 스텟 그대로 베끼기(데드카피)냐, 안 맞는 건 빼고 신규 메타 스텟 1개를 더하냐.

질문 ②

스킬 룰 구조

무기·장신구(자동·패시브) + 능동(처형·회피) + 처형/그로기 발동 효과는 어느 카테고리?

관통 원칙

린하게 + 정체성

뱀서의 강점은 '작은 스텟 셋'. 베끼되, 이 게임만의 부여(비틀대기)를 살린다.

02 · 세부 ① — 스텟: 뱀서를 린하게 베끼기

그대로 베낄 것 · 뺄 것

뱀서 실제 스텟이 게임에서판정
최대 체력 · 회복 · 방어력체력 있음 / 회복·방어는 약함✓ 채택(회복·방어 보강)
이동 속도그대로✓
피해량(Might)HP를 깎아 그로기로 → 사실상 '제압'✓ %로 정리(핵심)
공격 범위베기 범위 · 처형 반경 · 저스트 판정✓ 핵심(처형)
지속시간오버드라이브 · 그로기 · 탈진 시간✓ 핵심(부여)
쿨타임 · 투사체 수처형·회피 쿨 / 추가 베기·연속 처형✓ 그대로
부활 · 자석 · 행운 · 성장생존 · 젬 자석 · 카드 등급 · 레벨 속도✓ 채택
투사체 속도근접이라 투사체 없음(채찍처럼)✗ 제외 · 투사체 무기 추가 시 부활
탐욕(골드) · 저주상점 없음 / 난이도 다이얼⏸ 보류
새로고침 · 건너뛰기 · 지우기레벨업 카드 기능(스텟 아님)△ 기능으로 검토

→ 거의 다 데드카피. 안 맞는 것(투사체 속도·골드)만 빼고 저주는 보류. 피해량이 곧 제압이라 신규 스텟은 따로 없다.

03 · 세부 ② — 게이지 통합 (C안 채택)

별도 '제압' 게이지·스텟 삭제 → HP 하나로 통합

"HP 깎는 게 곧 그로기 만드는 거잖아" — 그 말이 맞다. 엘리트는 HP 0 = 그로기라, HP 막대가 이미 그로기 게이지다.

전 · 게이지 2개

HP 막대 + 제압 게이지

제압 게이지는 저스트 회피로만 차는 별도 막대. 같은 '그로기'를 두 번 만드는 중복.

후 · 게이지 1개

HP 막대(=그로기) 하나

저스트 회피 = HP를 큰 한 방으로 깎음 + 약점 2배. 연출·보상은 유지, 게이지만 통합.

피해량(Might)이 곧 제압이다 — 많이 깎을수록 빨리 그로기. → 신규 메타 스텟 없음. 린한 데드카피로 충분.
04 · 세부 ③ — 스킬 룰 구조

무기 · 장신구 · 능동 기술 · 각인 (4트랙)

뱀서 틀(무기·장신구)에 이 게임만의 능동(처형·회피)과 발동형 각인을 더한다.

자동

무기 (Weapon)

알아서 나가는 자동 베기 계열(회전 칼날·충격파). 레벨업·진화. = 휩쓸기 갈래.

패시브 수치

장신구 (Accessory)

항상 켜진 스텟 부여(위 스텟들). 무기 진화 재료도 겸함.

능동(누름)

능동 기술

직접 쓰는 스킬: 대시 베기·저스트 반격·화면 전체 처형(궁극).

발동형

각인 (Sigil)

조건 충족 시 자동 발동. "처형 시 폭발", "그로기 시 전파" 등. 처형·그로기 빌드의 심장.

진화: 무기 max + 장신구 → 진화 무기(뱀서식).  ·  칸 제한으로 빌드 갈래(처형·휩쓸기·몰아치기·회피)를 강제.

05 · 세부 ④ — 처형·그로기 빌드의 심장 = 각인

'처형 시 / 그로기 시' 발동형 각인으로 와우 빌드를 짠다

처형·그로기는 이 게임의 와우포인트. 그 순간에 자동으로 터지는 효과(각인)를 모으면 → 처형·그로기 특화 빌드 완성. (뱀서의 아르카나 같은 발동형 슬롯)

처형 시 · On Execute
  • 처형 시 주변 광역 폭발 → 잡몹 정리·연쇄 그로기
  • 처형마다 오버드라이브 추가 스택·지속↑
  • 처형 시 체력·보호막 회복 (처형 흡혈)
그로기 시 · On Groggy
  • 적 그로기 → 주변 적에 그로기 전파(연쇄)
  • 그로기 적 발밑 장판 생성
  • 그로기 적이 많을수록 피해↑ / 살짝 슬로우

각인끼리 시너지: 처형 → 폭발 → 연쇄 그로기 → 더 많은 처형 → 눈덩이. 이 발동형 루프가 곧 '와우 빌드'다.

06 · 결론 — 결정 & 체크리스트

린 데드카피 · 게이지 통합 · 4트랙(각인 추가)

스텟 = 뱀서 실제 셋 그대로(신규 0, 제압=피해량) · 게이지 HP 통합(C).
스킬 = 무기·장신구·능동 기술 + 각인(발동형) — 처형·그로기 와우 빌드의 심장.
