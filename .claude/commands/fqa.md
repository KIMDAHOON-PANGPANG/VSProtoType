VSQA 스킬을 사용해 **FQA 세션**을 진행하라.

- `QA/FQA/FUN_CRITERIA.md`(재미 기준 원장)를 먼저 읽고 시작한다.
- TQA 봇 배치 실행 → `QA/FQA/sessions/`에 새 판정 시트 생성 → 실플레이 가이드와 함께 안내하고 **판정을 기다린다** (판정 없이 밸런스를 바꾸지 않는다).
- 사용자가 이 명령과 함께 판정을 준 경우(예: "1번 ⭕ 3번 ❌ ...")에는 시트 기입 → 기준 승격 → 재조정 → 검증까지 진행한다.

사용자 추가 요청: $ARGUMENTS
