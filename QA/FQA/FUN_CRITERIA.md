# FUN_CRITERIA — 재미 기준 원장 (단일 소스)

> 사용자(기획자)의 재미 판정이 쌓여 만들어지는 문서. Claude 는 밸런스를 만질 때
> **반드시 이 문서를 먼저 읽고**, 각 기준의 "봇 지표 번역·목표 범위"를 TQA 배치로 검증한다.
>
> 상태: `제안`(Claude 초안 — 사용자 확정 대기) / `확정`(사용자 승인) / `보류` / `폐기`
> 사용자는 표를 직접 고쳐도 되고, 채팅으로 "F-01 확정, 범위는 좀 더 빡세게" 라고만 해도 된다.
